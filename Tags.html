TAGS

Paragraph:

The p tag in HTML is used to define a paragraph.

BOLD:

The b tag in HTML is used to make the text bold.

Italic:

The i tag in HTML is used to make the text italic.

Subscript:

The subscript tag in HTML is used to define subscript text. for example:

  • H2O
  • CO2

SuperScript

superscript The superscript tag in HTML is used to define superscript text.

hr:

The hr tag in HTML is used to create a horizontal line or horizontal rule.

Strong:

The strong tag in HTML is used to define strong importance or importance of strong emphasis, typically displayed as bold text.

EMPHASIS:

The em tag in HTML is used to define emphasized text, typically displayed as italicized.
